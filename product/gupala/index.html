Gupala

Pengamanan ekstra untuk motor kesayangan anda

Rp. 160.000

Pesan Sekarang

Lihat Video



Gupala adalah sebuah produk pengamanan motor dari tindakan pencurian, pembegalan, dan penggunaan yang tidak diinginkan. Dengan memakai produk gupala motor anda menjadi aman.

Bagaimana Gupala bekerja?

Tertanam di motor

Gupala akan dipasang ke motor anda dengan bantuan montir/teknisi yang handal. Dengan ukuran yang ideal maka mudah untuk dipasangkan ke motor anda.

Sandi Motor

Gupala menggunakan sandi motor hingga tiga kode kombinasi yang bisa diatur sesuai keinginan

Alarm Motor

Gupala memiliki varian yang bisa membunyikan alarm berupa klakson saat seseorang/pencuri mencoba menghidupkan motor anda tanpa memasukkan sandi dengan benar

Berbagai manfaat untuk anda

Anti Begal/Maling

Motor tidak akan menyala jika maling/begal tidak memasukkan kode dengan benar

Alarm

Motor akan membunyikan alarm saat motor dipaksakan untuk hidup

Sandi Motor

Motor bisa dihidupkan apabila sandi motor yang sudah diatur dimasukkan secara benar

Varian Gupala

Varian 1 ( 1 Code )

Merupakan varian yang paling ekonomis, tidak mengkonsumsi aki, mudah dalam pemasangan, compactible berbagai jenis motor, kode dapat dicustomisasi ketika pemasangan, telah diuji lebih dari satu tahun.

Fitur :

  • Satu kode kombinasi
  • Saklar ON-OFF sistem
  • Waterproof
  • Kabel SNI 0.75 cm
  • Tidak mengkonsumsi aki
Rp. 160.000,-
Varian 2 ( 2 Code )

Untuk anda yang suka kemudahan dalam penggunaan namun tetap mengutamakan keamanan, maka varian 2 merupakan alternatif untuk anda. Varian 2 menggunakan dua kode berurutan, menyulitkan pencuri untuk mengambil motor anda.

Fitur :

  • Dua kode kombinasi
  • Saklar ON-OFF sistem
  • Waterproof
  • Kabel SNI 0.75 cm
  • Tidak mengkonsumsi aki
Rp. 200.000,-
Varian 3 ( 3 Code )

Merupakan varian yang mempunyai kombinasi yang paling banyak dengan tiga kombinasi, tidak mengkonsumsi aki, mudah dalam pemasangan, compactible di berbagai jenis motor, kode dapat dicustomisasi ketika pemasangan.

Fitur :

  • Tiga kode kombinasi
  • Saklar ON-OFF sistem
  • Waterproof
  • Kabel SNI 0.75 cm
  • Tidak mengkonsumsi aki
Rp. 220.000,-
Varian 4 ( 1 Code + Trap )

Merupakan varian yang mempunyai satu kombinasi dan empat jebakan, pencuri , mudah dalam pemasangan, compactible berbagai jenis motor, kode dapat dicustomisasi ketika pemasangan.

Fitur :

  • Satu kode kombinasi
  • Empat kode jebakan
  • Saklar Reset
  • Saklar ON-OFF sistem
  • Waterproof
  • Kabel SNI 0.75 cm
  • Tidak mengkonsumsi aki
Rp. 240.000,-
Varian 5 ( 1 Code + Horn )

Varian dengan satu kode kombinasi dan notifikasi klakson, Varian 5 merupakan varian yang paling inovatif, jika anda salah memasukan kode maka klakson akan menyala


Fitur :

  • Satu kode kombinasi
  • Notifikasi klakson
  • Saklar ON-OFF klakson
  • Saklar ON-OFF sistem
  • Waterproof
  • Kabel SNI 0.75 cm
  • Tidak mengkonsumsi aki
Rp. 260.000,-
